Dashboard
This project was more about backend, making api calls and making documentation.

What I learned

Winscp

I was tasked to use Winscp to store all images and documents for the dashboard.

Api

Using laravel I created api calls for the frontend. one of my calls would save images to the sserver and store the path to the image in the DB which we could retrieve.

Swagger

Our apis needed documentation which is where swagger comes in, I used swagger to show all apis in an organised website.

DB

We needed DB tables to store users, documents, images and much more. i learned much about DB relationships.
